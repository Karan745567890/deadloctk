# Deadlock Detection and Prevention Simulator

A Python-based simulation tool for detecting and preventing deadlocks in operating systems.

## Features
- Banker's Algorithm for safety state detection
- Resource Allocation Graph (RAG)
- Prevention strategies: Resource Ordering, Hold & Wait Denial
- Real-time logging with GUI (Tkinter)

## Technologies Used
- Python
- Tkinter (GUI)

## How to Run
1. Clone the repo
2. Run: python os.py
